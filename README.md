# Excel_Basics

Some essential Excel skills that are useful for data cleaning and analysis. Here's what i performed for cleaning :

Practice-Data.xlsx :

    ✔ 1. Date Formatting
        Convert text dates into proper date format
        Change display format (DD/MM/YYYY, MMM-YY, etc.)
        Helps in sorting, filtering, and performing date-based calculations
    
    ✔ 2. Changing Column Format
    
        Currency Format
          Used to convert numeric values into monetary format
          Shortcut: Ctrl + Shift + $
        
        Percentage (%) Format
          Converts decimals into percentage
          Example: 0.25 → 25%
          Shortcut: Ctrl + Shift + %

Messy-Data.xlsx :

    ✔ 3. Relative & Absolute Referencing
    
        This is one of the most important Excel concepts.
        Relative Reference: Changes when formula is copied (e.g., B2 → B3)
        Absolute Reference: Stays fixed using $ (e.g., $A$1)
        Useful for fixed values like tax, GST, discount

Data-Transpose.xlsx :   

    ✔ 4. Converting Row Data into Column
        Used Paste Special → Transpose
        Shortcut: Ctrl + C → Right-click → Transpose
        Helpful when data is arranged incorrectly or needs restructuring.

Day-1-Capstone.xlsx :

    ✔ 5. Basic Functions Used
        SUM
        =SUM(B2:B10)
        
        MIN
        =MIN(B2:B10)
        
        MAX
        =MAX(B2:B10)
        
        AVERAGE
        =AVERAGE(B2:B10)
        
        COUNT
        Counts numbers only
        =COUNT(B2:B10)
        
        COUNTA
        Counts text + numbers
        =COUNTA(B2:B10)
